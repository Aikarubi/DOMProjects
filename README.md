
# **✨ Proyectos con el DOM**

Pequeños proyectos desarrollados en JavaScript para practicar la manipulación del DOM. Todos los proyectos fueron realizados siguiendo tutoriales de YouTube.

## **🚀 Instalación**

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Aikarubi/DOMProjects.git

2. Ejecuta el archivo **index.html** para visualizar el proyecto en tu navegador.
## **📜 Citas Aleatorias**

Este proyecto es un generador de citas aleatorias. Cada vez que haces clic en el botón, se muestra una cita nueva en la pantalla.

![App Screenshot](/JS%20PROYECTOS/DOM/citas-aleatorias/imagenes/DOM1.png)

## **🎨 Color Aleatorio**

Este proyecto genera colores aleatorios. Al hacer clic en el botón, el fondo de la página cambia a un color generado aleatoriamente.

![App Screenshot](/JS%20PROYECTOS/DOM/color-aleatorio/imagenes/DOM2.png)

## **⏱️ Cronómetro**

Un cronómetro sencillo que te permite iniciar, pausar y reiniciar el conteo del tiempo.

![App Screenshot](/JS%20PROYECTOS/DOM/cronometro/imagenes/DOM3.png)

## **🎛️ Color RGB**

Una herramienta de personalización de colores que permite a los usuarios seleccionar y visualizar colores utilizando valores RGB.

![App Screenshot](/JS%20PROYECTOS/DOM/input-color-rgb/imagenes/DOM4.png)

## **📝 To-Do List Simple**

Una lista de tareas sencilla donde los usuarios pueden crear nuevas tareas, marcarlas como completadas y borrarlas. Este proyecto es ideal para entender cómo manipular elementos del DOM de manera dinámica, permitiendo agregar y eliminar elementos de la lista.

![App Screenshot](/JS%20PROYECTOS/DOM/lista-de-tareas/imagenes/DOM5.png)

## **🍕 Mis Toppings**

Proyecto para practicar la manipulación del DOM, donde se puede seleccionar diferentes toppings para una pizza. El código JavaScript está completamente comentado para facilitar su comprensión.

![App Screenshot](/JS%20PROYECTOS/DOM/toppings-de-pizza/imagenes/DOM6.png)
## **🛠️ Tecnologías utilizadas**

**Frontend:** HTML5, CSS3, JavaScript
